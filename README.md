# Obsidian-Guide
For a long time, I felt like I was learning without really building anything.

I was studying computer science, but I wasn't developing the habits I wanted. My notes were scattered, I forgot what I learned, and every new topic felt like starting over.

A few days ago, I decided to change that.

Instead of only watching tutorials, I want to build a personal knowledge base that grows with me. Every concept I learn, every mistake I make, and every project I build will have its own notes.

That's why I started learning **Obsidian**.

The main reason I chose it first is simple: it works offline. My internet connection isn't always available, so I wanted a tool that wouldn't stop me from learning.

So far, one feature has really stood out to me: **Backlinks**. I love the idea of connecting notes together instead of keeping them isolated. It feels like building a network of knowledge rather than collecting random documents.

I also customized the interface before I even started learning. 😄 I enjoy making my workspace feel comfortable because I'm planning to spend a lot of time in it.

At the moment, I haven't decided whether Obsidian will be my long-term tool. I also plan to learn **Notion** before making that decision. Maybe I'll end up using one of them, or maybe both for different purposes.

This is only the beginning of my journey.

From now on, I'll document what I learn about web development, algorithms, data structures, and every new skill I pick up. My goal isn't just to remember what I learn—it's to create a knowledge base that I can keep improving over the years and hopefully share with others.

For those of you who have used both:

**If you had to choose, would you recommend Obsidian, Notion, or using both together? I'd love to hear about your experience and why.**
